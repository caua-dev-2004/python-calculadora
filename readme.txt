# README do Projeto Lista de Afazeres

## Descrição

Este projeto é uma aplicação de lista de afazeres desenvolvida em Python utilizando a biblioteca Tkinter para a interface gráfica e SQLite para o gerenciamento de dados. O objetivo é permitir que os usuários adicionem, atualizem e removam tarefas de uma lista de afazeres de forma simples e intuitiva.

## Funcionalidades

- **Adicionar Tarefas**: Permite ao usuário inserir novas tarefas na lista.
- **Atualizar Tarefas**: O usuário pode modificar o nome de uma tarefa existente.
- **Remover Tarefas**: O usuário pode excluir tarefas da lista.
- **Exibição de Tarefas**: Mostra todas as tarefas cadastradas em uma lista.

## Tecnologias Utilizadas

- **Python**: Linguagem de programação utilizada para o desenvolvimento.
- **Tkinter**: Biblioteca para criação da interface gráfica.
- **SQLite**: Banco de dados utilizado para armazenar as tarefas.
