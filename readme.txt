# **README - Calculadora Python**

## **Descrição do Projeto**
Este projeto é uma calculadora simples desenvolvida em Python utilizando a biblioteca Tkinter para a interface gráfica. A calculadora permite realizar operações básicas como adição, subtração, multiplicação, divisão e cálculo de porcentagens.

## **Funcionalidades**
- **Operações Básicas**: Realiza adição, subtração, multiplicação e divisão.
- **Cálculo de Porcentagem**: Permite calcular porcentagens diretamente.
- **Limpar Tela**: Possui um botão para limpar a entrada atual.
- **Interface Gráfica**: Utiliza Tkinter para uma interface amigável e intuitiva.

## **Tecnologias Utilizadas**
- **Python**: Linguagem de programação utilizada para o desenvolvimento.
- **Tkinter**: Biblioteca padrão do Python para criação de interfaces gráficas.
